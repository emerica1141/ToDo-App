About this project

Here at this study project, I have delved into the fascinating realms of Python and Flask, meticulously honing my skills in database queries, CRUD operations, and the intricacies of Flask functionalities. This journey involves a hands-on exploration of HTML, TailwindCSS, and Flowbite components, weaving them together to create a useful learning project.

Functionalities:

  1. Registration and login authentication
  2. Task adding functionality
  3. Task editing and deleting
  4. Task "Done", which will remove task from home page and add it to Archive page
  5. Possibility to "Undo" task in Archives and bring it back to active phase in Home page
  6. Possibility to "Delete" task in Archive page
  7. Relationships between accounts, to-do list and archive tables, which allows application to show the owner of tasks
  8. "Reminder" functionality, which will send and email to accounts

Functionalities to add:

  3. Functionality to ask for confirmation when deleting tasks from Home and Archive
